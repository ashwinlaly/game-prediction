# game-prediction

    Back end service for Game Score and Prediction.

Modules Used:
    1. Express <br/>
    2. Mongoose <br/>
    3. body-parser <br/>
    4. socket.io <br/>
    5. chalkjs (Optional) <br/>

Procedures:
    1. Clone App
    2. cd game-prediction
	3. npm i
	4. node index.js
	5. import API Endpoints from /api - rest client
